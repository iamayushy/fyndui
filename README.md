# fyndui
## Installing the ui 
```
npm i fyndui
```

## importing in vue
```vue
<script>
import {ButtonVue} from "fyndui"
import "fyndui/dist/fyndui.css" 
export default {
  components: {
  ButtonVue
  }
}
</script>
<template>
<div>
<ButtonVue></ButtonVue> 
</div>
</template>

```
